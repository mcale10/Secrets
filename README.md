# Welcome to Secrets App

A simple secrets app built using Node.js, passport.js, oAuth2.0 and ejs.

## Install

```bash
# install requirements
npm install

# install dotenv 
npm i dotenv
Add to .env file
- MONGODB_URI - mongodb server link
- SECRET

Google 
- CLIENT_ID
- CLIENT_SECRET

Facebook
- FACEBOOK_APP_ID
- FACEBOOK_APP_SECRET
```

## Run server

```bash
# run app
nodem app.js
```

## Open app

Your app will now be accessible at localhost:3000
